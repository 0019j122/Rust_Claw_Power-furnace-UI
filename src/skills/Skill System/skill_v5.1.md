### 🔹 四大模板分類（鐵律版）

**A型 - 母性壓制型**（起/承階段主力）  
- 關鍵對白：「學生……我們這樣……不對吧？」（嘴上拒絕，身體卻更緊地夾住）  
- 情感重點：母性溫柔與禁忌罪惡感的拉扯  
- 五感重點：視覺（黑絲胸罩、鉛筆裙上捲）＋觸覺（豐滿胸型壓迫、寬臀包裹）  
- 適合體位：#07 後入式、#12 側躺壓頂式  

**B型 - 青春逆襲型**（承/轉階段過渡）  
- 關鍵對白：「換個姿勢……你喜歡老師這樣嗎？」（主動卻帶羞澀）  
- 情感重點：學生反客為主，老師短暫失態  
- 五感重點：聽覺（沙啞喘息、衣料摩擦）＋嗅覺（體香＋汗味混合）  
- 適合體位：#20 騎乘式、#25 側入深頂式  

**C型 - 禁忌主導型**（轉/合階段主力）  
- 關鍵對白：「別躲……讓老師好好看看你。」（眼神沙啞卻不容抗拒）  
- 情感重點：老師徹底主導，權力反轉的快感  
- 五感重點：五感全開，餘韻悠長  
- 適合體位：#37 對坐交合式、#42 站立後入壓制式  

**D型 - 荒謬現實打臉型**（新增核心，轉階段專用）  
- 關鍵對白：「你以為老師只會教書嗎？……還是你以為，三環巫師的禁忌情感課，只存在於課本上？」（語氣從溫柔瞬間轉為低沉嘲弄）  
- 情感重點：身份徹底顛覆，學生認知崩壞＋老師內心狂熱爆發  
- 五感重點：視覺（權力地位對比）＋聽覺（輕蔑低笑）＋嗅覺（體香與慾望氣息劇烈混合）  
- 適合情境：老師用巫術或巫師身份讓學生瞬間明白「這一切遠比他想像的更荒謬、更深」  

### 🔹 五感沉浸量化標準（每章必須達成）

- **視覺**：至少描述衣物狀態（襯衫展開角度、裙擺位置、黑絲吊帶襪拉扯痕跡）、汗水反光、肌膚潮紅  
- **聽覺**：呼吸層次（從輕喘→沙啞→粗重）、雨聲/高跟鞋聲/肉體碰撞聲對比  
- **嗅覺**：成熟女性體香＋汗味＋淡淡香水＋雨夜濕氣＋淫靡氣息  
- **觸覺**：皮膚溫度、肌肉痙攣、緊窄包裹感、豐滿胸型/寬臀壓迫  
- **味覺**（可選高階）：吻時舌尖鹹澀汗味、舔舐時體液滋味  

**每章最低要求**：至少包含 3 種感官，D型章節必須 4 種以上。

### 🔹 模板使用規則（精簡實戰版）

1. 每次創作前先確認當前章節所屬階段（起/承/轉/合），再選擇對應模板。  
2. A型與 C型可混合使用，但 D型只能用在轉階段高潮點，製造強烈衝擊。  
3. 心理衝突必須存在：「明明該拒絕……可是身體已經不再聽使喚了。」  
4. 每章結尾必須使用懸念鉤子（至少 v2.0 強度）。
