### 🔹 執行流程（三階段閉環）

**1. 初稿階段（快速產出，15-25分鐘）**
1. 確認當前章節所屬波浪階段（起/承/轉/合）
2. 選擇對應模板（A/B/C/D型）與推薦四十八手
3. 套用 V6.1 結構圖表，直接填入環境、時間、感官細節
4. 強制加入至少一次「明明該……可是……」心理衝突

**2. 精修階段（深化情感與張力，20-30分鐘）**
1. 檢查波浪節奏是否明顯起伏
2. 強化心理抵抗與身體誠實的反差
3. 確保粗暴動作不崩壞「端莊冷艷教師」氣質（所有粗暴必須包裹在禁忌、羞恥、巫師身份反轉之中）
4. 五感描寫密度達標，並加入 D型荒謬打臉（若為轉階段）

**3. 最終審查階段（完成前必做）**

### 🔹 最終審查清單（7項鐵律，全部打勾才算通過）

- [ ] **角色一致性**：去掉名字後，仍能透過語氣、動作、穿著（黑絲胸罩、鉛筆裙、吊帶襪）辨識出是「端莊冷艷三環巫師女教師」  
- [ ] **禁忌衝突**：至少包含一次言語抵抗（「我們這樣……不對吧？」）與身體誠實的反差  
- [ ] **波浪節奏**：情緒有明顯高低起伏，性張力逐步上升，不平鋪直敘  
- [ ] **五感沉浸**：視覺≥3、聽覺≥2、嗅覺≥2、觸覺≥2（轉階段必須五感全開）  
- [ ] **模板與四十八手匹配**：符合 V6.1 對照表，D型僅用於轉階段高潮點  
- [ ] **荒謬現實平衡**：粗暴或權力反轉必須服務於「師生禁忌」與「巫師身份顛覆」，不能單純變成強姦犯發洩  
- [ ] **懸念鉤子**：結尾至少達到 v2.0 強度（讓讀者強烈期待下一章），禁止重複預告  

**總結建議（每次創作前默念）**  
1. 嚴格按照當前階段選擇模板與體位，避免重複。  
2. 每章必須保留「端莊外表 vs 內心狂熱」的核心反差。  
3. 章節間要有情節推進（巫術、師徒關係、島嶼/世界觀線索），不能只是重複性愛。  
4. 粗暴必須帶有心理層面的「打臉」與「崩壞」，才能維持老師的冷艷氣質。
