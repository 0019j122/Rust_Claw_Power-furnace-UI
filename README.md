## UI 背景設計

### 方案A：全景背景（The Atmospheric Backdrop）

為了讓AI（如Midjourney）既符合你宏大世界觀，又不會讓使用者看不清楚文字的背景圖。

🚀 專業級提示詞 (The Master Prompt)

建議使用的提示詞（複製和貼上）：

```
像素藝術，1280x720，令人嘆為觀止的電影級側視圖，展現了一座巨大的垂直魔法能量熔爐，它坐落於一片黑暗深邃的火山谷中。這座結構龐大而層疊，隱約勾勒出社會階層的輪廓：鬱鬱蔥蔥的農業區頂部、繁華熱鬧的中層城市以及陰暗的工業/岩漿區底部。熾熱的岩漿在底部奔湧。畫面採用電影級光照，邊緣對比度高，營造出強烈的大氣深度感。關鍵在於：中心區域被設計成低對比、柔和且富有氛圍感的負空間，以確保使用者介面清晰易讀​​。深邃的陰影、橙色的熔岩光暈、微妙的魔法藍色生物發光、精細的16位細節，營造出史詩般的黑暗奇幻氛圍，是一款專業的遊戲背景素材。
```

🔍 為什麼這樣設計？

為了讓你以後能夠自行調節，

**視覺風格與解析度（Style & Spec）：**

- 像素藝術，16 位元細節：定義了您要求的像素風格
- 1280x720：設定構圖比例。

**世界觀建構（World Building）：**

- 分層巨型結構/社會層輪廓: 我沒有叫AI「剪影」（側影），這使得AI能夠使用
- 火山谷/岩漿流: 建立黑暗、岩漿、巫師動力

**UI 核心優化（UI 秘密武器 - 最重要！）：**

- 關鍵點：中心區域設計成低對比、柔和且富有氛圍感的負空間。：這句話是給 AI 的「救命指令」。它要求背景才不會搶走文字的焦點。

**色彩管理（色彩理論）：**

- 橙色熔岩光芒對"Mag"神奇的藍色生物發光: 利用「補色」（橘子）

💡 應用建議

當你拿到圖片時，

**使用遮光罩（Overlay）：**
不要直接放圖

```css
.ui-background {
  background: linear-gradient(rgba(0,0,0,0.3), rgba(0,0,0,0.3)), url('your-pixel-art.png');
  background-size: cover;
}
```
