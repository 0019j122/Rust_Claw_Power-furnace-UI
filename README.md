# [龍蝦_動力爐]


龍蝦_動力爐是一個基於Rust語言開發的AI自主學習迭代進化,像素遊戲環境,角色扮演故事形式,趣味又不會無聊,AI輔助文學創作工具，旨在幫助作家和創作者提升寫作效率和創造力。它結合了多位知能體、記憶庫和技能系統，提供了一個強大的平台來生成、優化和管理文學作品.

## 功能特性

- AI : 協助文學創作、學習、升級進化
- 多位知能體: 角色扮演、記憶庫、技能系統
- 參考、攝取、合成、融合、提煉 = 動力爐器具 //出處想法是巫師的煉金爐，將各種元素融合成新的作品
- 記憶
- 自動章節生成：根據情節大綱快速產生章節內容
- 角色對話優化：改進角色互動和對話自然度
- 文風統一：保持整部作品的敘述風格一致
- 創意輔助：提供寫作建議和劇情發展方向
- 批量編輯：同時處理多個文件和章節
- 反思 : 好玩、好笑、有趣、足夠色情{肉麻、變態、偷窺、粗暴、強奸}、好無聊?不好看?不知道演什麼?
- 迭代 : 更新
